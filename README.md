# streamlit-kalkulacka-podpory

### Streamlit aplikace: Kalkulačka podpory v nezaměstnanosti

V aplikaci uživatel zadá věk a příjem. Na základě zadaných údajů a pravidel zakódovaných v Pythonu se vypočítá podpora v nezaměstnanosti v roce 2026 a pro srovnání také v roce 2025. Hodnoty jsou pro lepší představu vizualizované v grafu.

<ins>Obsahuje:<ins>
- Streamlit elementy (zadání hodnot, tlačítko, sloupcový graf, text)
- vytvořené funkce v Pythonu pro výpočet podpory (podmínky, iterace, práce se slovníky)

<ins>Použité technologie:<ins>
- **Python: streamlit, pandas**

<ins>Aplikace na produkci:<ins>
- https://kalkulacka-podpory.streamlit.app/
